Installing the modpack:

Launcher

This is the primary way to install the modpack. The launcher allows for full control of mods, and allows users to pick optional mods they may want to have.

Download Launcher

How to use the launcher:

  1. Download and install x64 Java 8 if you do not already have it installed. You must have this installed for the launcher to work correctly.
  2. Download the correct version of the launcher for your operating system. (.exe is for Windows users, and the .jar is for Linux and OSX users)
  3. The launcher will download all needed contents to your Documents folder with the name of ‘ArdaCraft Launcher’
  4. Run the ArdaCraft launcher, and select the ArdaCraft Modpack tab on the left of the program.
  5. Log into your Minecraft Account.
  6. Select your desired optional mods, and click ‘Ok’
  7. The game may take several minutes to start, and your game may freeze. If this happens let it load, and eventually it will go to the main menu.
  8. You will not have the resourcepacks loaded the first time you load the game using the launcher. You will need to enable the resourcepack, and any of the optional mod resourcepacks.
  9. Once everything is setup, log into the server with the ip ‘mc.ardacraft.me’

Installer

This is a program that will download all the needed mods, and create a new profile in the default Minecraft launcher. This only comes with the necessary mods to join the server, and does not have the optional mods the launcher comes with.

Download Installer

How to use the installer:

  1. Make sure you have Java 8 installed first.
  2. Download and run the installer (double click the jar file).
  3. If you do not already have Forge for minecraft-1.8.9, you will be prompted to install it before continuing.
  4. Select the 'Install' option and press 'Run', then wait until the installer says it has finished.
  5. Open the minecraft launcher and select the ArdaCraft profile before running the game.

Self-Install

This is not the recommended way to join the server, but if you are having troubles with either of the two options above, you can install everything yourself. This only comes with the necessary mods to join the server, and does not have the optional mods the launcher comes with.

Download Self-Install Files

How to self-install:

  1. Install the recommended version of Forge for 1.8.9.
  2. Download the zip file.
  3. Drag the zip file’s contents to your .minecraft folder.
  4. Launch Minecraft with the forge profile enabled.
  5. Connect to the server with the ip ‘mc.ardacraft.me’
